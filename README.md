Кастомная интеграция кондиционера GoldStar GSAC/GSACI в Home Assistant с помощью самодельного свистка Hommyn из esp32-c3.
![orig](https://github.com/user-attachments/assets/d60b3993-ae8f-4545-a4be-71e95870c672)

Для начала необходимо заиметь (купить или самому спаять) свисток Hommyn. 
Покупается в любом маркетплейсе, паяется из ESP32-c3 и преобразователя уровней (можно и без него, просто добавив пару резисторов).
Более подробно можно прочитать тут - https://4pda.to/forum/index.php?showtopic=1076299&st=440#entry137049462
