Кастомная интеграция кондиционера GoldStar GSAC/GSACI в Home Assistant с помощью самодельного свистка Hommyn из esp32-c3. 
![246cadceedc56557afbd2a97586df4c5234e98b59bbb879a788ec9ea96baed20](https://github.com/user-attachments/assets/f5316980-86d0-4355-ba43-45d4ac0d0a15)

1. Для начала необходимо заиметь (купить или самому спаять) свисток Hommyn. 
Покупается в любом маркетплейсе, паяется из ESP32-c3 и преобразователя уровней (можно и без него, просто добавив пару резисторов).
Более подробно можно прочитать тут - https://4pda.to/forum/index.php?showtopic=1076299&st=440#entry137049462

2. Далее необходимо прописать наше устройство в приложении Hommyn и там же в настройках устрйоства указать локальный адрес сервера mqtt.
Если у вас установлен HAOS и Mosquitto, то адрес совпадает с адресом HAOS, порт 1883, логин/пароль известны только вам))).

3. Выключаете кондиционер из сети, через минуту-другую включаете и смотрите что происходит в mqtt-брокере. У вас должен появится топик, состоящий из 12 букв/цифр.
Обычно это mac-адрес esp32-c3
